# 海洋發聲者 (Ocean Voice) - 海洋生物與生態保育宣導

這是「網頁前端程式設計」的期末專題作品。本網站旨在透過互動體驗，提升大眾對海洋保育議題（瀕危生物、珊瑚白化）的認識，響應聯合國永續發展目標 SDG 14。

## 📂 專案檔案結構說明

本專案包含以下主要檔案：

* `index.html`: 網站首頁 (包含專案介紹與導覽)
* `map.html`: 瀕危生物地圖 (使用 Leaflet.js 互動地圖)
* `coral.html`: 珊瑚白化模擬 
* `quiz.html`: 海洋知識行動測驗
* `style.css`: 全站共用樣式表 
* `script.js`: 核心互動邏輯 
* `README.md`: 專案說明文件 (本檔案)

## 🚀 如何執行 (How to Run)

1.  請確保資料夾中包含所有 `.html`, `.css`, `.js` 以及圖片檔案。
2.  直接雙擊 `index.html` 使用瀏覽器 (Chrome/Edge) 開啟即可瀏覽。
3.  建議使用 F12 開發者工具切換至手機模式，體驗 RWD 響應式設計。

## 🛠️ 使用技術

* **HTML5 / CSS3**: 網頁結構與視覺動畫
* **JavaScript (ES6)**: DOM 操作與邏輯運算
* **Leaflet.js**: 開源地圖函式庫
